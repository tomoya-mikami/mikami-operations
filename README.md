# mikami-operations

三神知也の事業運営・AI作業引き継ぎ用ワークスペース。

## 目的

このリポジトリは、Codex・Claude・ノートPC・デスクトップPC・スマホ間で作業内容を引き継ぐための共通作業台です。

## 運用ルール

- 結論先出しで記録する
- 作業の節目で `handoff.md` を更新する
- 進行中の事業や案件は `projects.md` にまとめる
- 機密情報は記録しない
- Google Drive上の削除・移動は必ず三神さんの許可を取る
- 会話内容の保存については毎回許可を求めなくてよい

## 使い分け

- GitHub: 作業ログ、AI引き継ぎ、プロジェクト台帳、Markdownメモ
- Google Drive: PDF、Excel、Word、請求書、融資資料、画像、重要資料
- ローカルPC: 実作業、検証、一時ファイル

## 主要ファイル

- `README.md`: このワークスペースの説明
- `handoff.md`: AI間・PC間の作業引き継ぎ
- `projects.md`: 現在のプロジェクト一覧
