# カーパーツリサーチ作業マニュアル

最終更新: 2026-05-27

## 結論

この作業の目的は、eBayで売れそうなカーパーツを見つけ、仕入れ可能か、赤字にならないか、送料で損しないかを確認して、出品候補として正しくシートへ入力すること。

初めてのパートさんは、まず「型番を正確に扱う」「モノタロウ価格と入数を間違えない」「基準セラーの総額を下回らない」「分からないものは無理に判断しない」の4つを守ればよい。

## 作業の全体像

1. 基準店舗またはライバルセラーの商品一覧を見る。
2. 商品タイトルや説明から型番を探す。
3. 型番をモノタロウで検索する。
4. 同じ商品だと確認できたモノタロウURLをB列に入れる。
5. モノタロウの税別価格をC列に入れる。
6. 1袋に何個入っているかをD列に入れる。
7. E列の想定仕入価格を確認する。
8. eBay基準店舗で同じ型番の出品があるか確認する。
9. 基準店舗URLをH列に入れる。
10. 価格、送料、利益、サイズ/重量の不安があれば要確認にする。

## 作業前チェック

| 確認 | 内容 |
|---|---|
| 商品カテゴリ | カーパーツ、車種関連部品である |
| 型番 | メーカー純正品番または部品番号が読める |
| 仕入先 | モノタロウで同じ型番の商品ページが見つかる |
| 価格 | 税込ではなく税別価格を確認する |
| 入数 | 1個入り、2個入り、左右セットなどを確認する |
| 状態 | 在庫切れ、取扱終了、納期未定ではないか確認する |
| 競合 | 基準店舗または他セラーの価格が確認できる |
| 送料 | 大きい、重い、長い、割れ物の可能性があれば要確認 |

## シート入力ルール

| 列 | 入力内容 | 注意点 |
|---|---|---|
| B | モノタロウ商品URL | 検索結果ページではなく商品ページURLを入れる |
| C | モノタロウ税別価格 | 税込価格と間違えない |
| D | 入数 | 1袋2個入りなら2、1個なら1 |
| E | 想定仕入価格 | 自動計算。原則触らない |
| F/G | ヤフオク等の参考価格 | 指示がある場合だけ入力 |
| H | eBay基準店舗URL | 同じ型番の出品ページURLを入れる |
| メモ欄 | 要確認事項 | 不安があるものは必ず書く |

## 型番確認のコツ

| よい状態 | 要確認 |
|---|---|
| 型番が完全一致している | 末尾記号だけ違う |
| メーカー名も一致している | 互換品、社外品、代替品と書いてある |
| 商品画像と形状が近い | 画像が明らかに違う |
| 入数が商品説明と一致している | 左右セット、2個入り、1台分などが曖昧 |
| 取扱中で価格が表示されている | 取扱終了、在庫切れ、納期未定 |

迷ったら出品候補にせず、要確認にする。

## 価格設定の考え方

基準セラーが同じ型番を出品している場合:

- 基準セラーの `商品価格 + 送料` を確認する。
- 三神さん側の `商品価格 + 送料` が、その総額を下回らないようにする。
- 商品価格と送料の内訳は完全一致でなくてもよい。
- 総額を下回ると価格秩序が崩れるため注意する。

基準セラーが出品していない場合:

- 他セラーの価格を見る。
- eBay Product Research / Terapeak で売れた価格を確認できるとよい。
- 利益率が低い場合は無理に出品候補にしない。
- 高額、重い、大きい、不明な商品は要確認。

## NG例

| NG | 理由 |
|---|---|
| 型番が似ているだけで同一扱いにする | 適合違いで返品・クレームにつながる |
| 税込価格をC列に入れる | 仕入計算がズレる |
| 2個入りを1個として入力する | 価格・利益計算がズレる |
| 在庫切れや取扱終了を見落とす | 売れた後に仕入れできない |
| サイズ/重量が不明なのに安いShipping Policyを選ぶ | 送料赤字になる |
| 基準セラー総額より安くする | 価格秩序が崩れる |
| 分からないまま空欄で進める | 後で確認できない |

## 要確認にする基準

| 状況 | メモ例 |
|---|---|
| 型番が完全一致しない | 型番末尾違い 要確認 |
| 入数が分からない | 入数不明 |
| 取扱終了/在庫切れ | 取扱終了候補 |
| 納期未定 | 納期未定 |
| 重そう/大きそう | 送料要確認 |
| ガラス、液体、危険物、エアバッグ系 | 出品可否要確認 |
| 基準セラーが見つからない | 基準セラーなし |
| 利益が薄そう | 利益要確認 |

## 作業者のゴール

1日のゴールは、件数だけではなく「あとで出品判断できる品質のデータ」を残すこと。

優先順位:

1. 型番とURLの正確さ。
2. 税別価格と入数の正確さ。
3. 取扱終了・在庫切れの見落とし防止。
4. 基準セラー総額の確認。
5. 送料・サイズ不安のメモ。

## 参照した公式情報

- eBay Seller Hub Reports: CSV/Excelテンプレートで出品追加、修正、再出品、終了が可能。
- eBay Bulk Listing Tools: Seller Hub Reports、MIP、Trading API、Sell Feed APIなどの一括管理手段がある。
- eBay Product Research: Seller Hub Researchから販売トレンド、平均販売価格、送料、Sell-throughなどを確認できる。
- eBay Motors Parts Compatibility: カーパーツでは適合情報/Fitmentが検索性と返品リスクに影響する。
