# 業務ツール棚卸し台帳

最終更新: 2026-05-23

## 目的

これまで複数チャット・複数AI・複数スプレッドシートで作成した業務ツールを棚卸しし、重複・不足・廃止候補を整理する。

最終目的:

- 必要な機能を再設計する
- 不要/重複機能を廃止する
- ジャンル別に整理する
- 再構築対象を決める
- 将来的に管理ダッシュボード化する
