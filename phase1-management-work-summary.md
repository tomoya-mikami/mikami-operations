## これからの課題

### 利益が見える経営管理

| 課題 | 必要なこと |
|---|---|
| 実質利益が見えない | 売上、仕入、eBay手数料、送料、関税、外注費、固定費を月別に並べる |
| 固定費が見えにくい | メルカード、請求書、サブスクを分類する |
| 商品カテゴリ別の利益が見えない | SKU/商品名からカテゴリを付ける |
| 送料・関税の影響が見えない | eBayインボイス、配送請求、送料マスターを接続する |
| 月次締め状態が見えない | データ取込状況チェックを作る |

### Phase1安全運用

| 課題 | 必要なこと |
|---|---|
| GASエディタで関数を探す運用が難しい | スプレッドシート上のメニュー/確認画面を作る |
| 実行してよい処理と危険処理が混ざっている | 操作区分を明確化する |
| 移動・リネームが怖い | 必ずドライランを先に出す |
| ログが見えない | 実行ログシートを作る |
| 月次で何を見ればいいか分からない | 月次チェックリストを作る |
