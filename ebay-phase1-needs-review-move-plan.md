# eBay Phase1 `00_要確認` 移動予定リスト案

作成日: 2026-05-26

## 結論

`00_要確認` のファイルは、すぐ移動しない。

まずはファイル名をそのまま保存せず、分類・件数・移動先候補だけで確認する。三神さん確認後に、必要ならマスク済みの個別リストを作り、さらにその後に手動移動またはドライラン付きGASへ進む。

## 今回の前提

- Google Drive内の削除、移動、リネームはしていない。
- ファイル本文、PDF本文、画像本文、ZIP中身は開いていない。
- 個人名、メールアドレス、請求書番号、ファイルIDは保存していない。
- 件数はブラウザで見えた範囲の50件ベース。

## 移動候補の考え方

| 分類 | 件数 | 移動先候補 | すぐ移動できるか | 理由 |
|---|---:|---|---|---|
| FedEx請求書 | 14 | 月別フォルダの `各種請求書` または `送料/配送費` | まだ不可 | 保存先カテゴリ名を決める必要あり |
| サブスク/サービス系請求書・領収書 | 14 | 月別フォルダの `サブスク` または `各種請求書` | まだ不可 | サブスクと単発請求の分け方確認が必要 |
| 配送/通関/納品ラベル系 | 5 | 月別の `配送/通関`、または提出不要候補 | まだ不可 | 税理士提出対象か確認が必要 |
| CSV/Excel配送・返送・フルフィルメント系 | 6 | eBay配送書類系、または保管専用 | まだ不可 | CSV/Excelを提出するか確認が必要 |
| 振込明細 | 3 | 月別の `支払/振込明細` | まだ不可 | 月別フォルダに該当カテゴリがあるか確認が必要 |
| Refund/CreditNote | 1 | `返金/クレジットノート` または `各種請求書` | まだ不可 | 請求書と分けるか確認が必要 |
| 画像ノイズ候補 | 1 | 保存対象外候補 | まだ不可 | 削除は三神さん確認後。まず除外候補に置く |
| ZIP | 1 | 保留 | 不可 | 中身確認の許可が必要 |
| その他 | 5 | 保留 | 不可 | ファイル名だけでは判断不可 |

## 推奨する最終フォルダルール

| 書類 | 推奨フォルダ |
|---|---|
| eBay売上・レポート | `04_税理士提出用/YYYY年MM月/eBay売上` |
| eBayインボイス | `03_eBayインボイス/YYYY年MM月` または既存月フォルダ直下 |
| FedEx/DHL等の配送請求 | `04_税理士提出用/YYYY年MM月/各種請求書` |
| サブスク | `04_税理士提出用/YYYY年MM月/サブスク` |
| カード明細 | `04_税理士提出用/YYYY年MM月/カード明細` |
| 外注請求書 | `04_税理士提出用/YYYY年MM月/外注請求書` |
| 返金・CreditNote | `04_税理士提出用/YYYY年MM月/返金・調整` |
| 提出不要候補 | すぐ削除せず、`00_要確認` 内で別ラベル管理 |

## 先に決めたいこと

| 決めること | 推奨 |
|---|---|
| FedEx請求書の保存先 | `各種請求書` へ入れ、ダッシュボードでは送料/配送費として分類 |
| 配送/通関ラベルの扱い | 税理士提出が必要か確認するまで保留 |
| CSV/Excelの扱い | 元データ保管が必要なら保管、提出不要なら削除候補へ |
| Refund/CreditNote | 通常請求書とは分けて `返金・調整` を作る |
| メール装飾画像 | 保存対象外候補。ただし削除は確認後 |
| 年フォルダ表記 | `YYYY年MM月` の月フォルダ直置きを正とする案が自然 |
